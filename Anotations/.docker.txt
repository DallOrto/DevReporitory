===================>> COMANDOS  <<===================
    > docker ps 
        - visualização de todos containers que estão  inicializados.
    > docker ps -a 
        - visualização de todos containers, tanto  inicializados, quanto parados.
    > docker rm (id do container)
        - para remoção de um container específico. OBS: Só é possível executar remoção de um container "parado". Ou seja, não inicializado.
    > docker start (id do container)
        -para iniciar um container específico.
    > docker stop (id do container)
        -para parar um container específico.
